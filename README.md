# sample-thymeleaf-todo
Sample Todo application with Spring Boot and Thymeleaf.

#### Application
- Java 11, Kotlin 1.6
- Spring Boot 2.6
- Spring Security 5.6.3
- Thymeleaf, Bootstrap
- MySQL 5.7
- Mybatis 3.5.9

#### Infrastructure
- Docker Compose
- Buildpack

#### Testing, CI/CD
- JUnit
- Testcontainers
- ArchUnit
- Jacoco
- Codecov
- GitHub Action

# Usage
## build application container
```
make build
```

## start
```
make up
```

## down 
```
make down
```

## start db
```
make db-up
```
